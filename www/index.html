Angular Dynamic Hooks - Standalone

This is a normal index.html file that is NOT generated by Angular. Nor is there an Angular app anywhere. This is simply standard HTML that could come from anywhere.

Standalone mode then allows us to mount Angular components directly into such arbitrary HTML.

The components are found by their hooks (as usual).

In this case, parseHooks is called from a normal Typescript setup and manually bundled with Webpack along with other Typescript code.The Angular CLI is NOT used at all to build it.
